Java基础梳理：泛型

发表于 2015-08-18   |   分类于 Java   |   阅读次数

泛型的作用

泛型将类型检查提前到编译期，从而避免了运行时可能引发的 ClassCastException异常；此外，泛型消除了类型转换，使代码更简洁。

泛型占位符

占位符根据需要，可以有一个或者多个，多个时尖括号之间放置一个逗号分隔。可以是单个或多个大写字母，用于指示所定义的参数类型。如下列举一些惯用的占位符：
E：元素
K：键
N：数字
T：类型
V：值

有界类型

有界类型 在类型参数部分指定 extends 或 super 关键字，分别用上限或下限限制类型，从而限制泛型类型的边界。

泛型通配符

无界通配：?
子类限定：? extends Object
父类限定：? super Integer

泛型擦除

Java的泛型是伪泛型。泛型只在编译期有效，编译后字节码不存在泛型信息
